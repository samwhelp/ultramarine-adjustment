## Usage


### install

run

``` sh
./install.sh
```

or run

``` sh
make install
```


### package-install

run

``` sh
./package-install.sh
```

``` sh
make package-install
```

``` sh
./asset-install.sh
```

or run

``` sh
make asset-install
```

``` sh
./prototype-install.sh
```

or run

``` sh
make prototype-install
```
